# Marking Up a Letter

A semantic HTML and CSS project based on the MDN Web Docs "Marking Up a Letter" exercise.

## 📌 Objective

The objective of this project is to convert a plain text letter into a well-structured HTML document using semantic HTML elements and basic CSS styling.

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📁 Project Structure

```
│── index.html

│── README.md
```
